# House Price Prediction
Dictionary req : scikit-learn , pandas 

Description:
This is a simple machine learning project to predict house prices.  
We use two models:
- Decision Tree
- Random Forest

-data :
- Number of rooms
- Number of bathrooms
- Land size
- Building area
- Year built
- Latitude and Longitude

split the data into training and validation sets.  
Then train the models and check accuracy using Mean Absolute Error (MAE)
